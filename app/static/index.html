LangChain Agent
LangChain Multi-Tool Agent
What it is
A ReAct agent built with LangChain and LangGraph — it reasons through your question, decides which tools to use, runs them, and synthesizes the result into a response.
Tools available
🔍 Web Search — live DuckDuckGo results 🧮 Calculator — math expressions 🕐 Date & Time — current time in any timezone
Memory
The agent remembers your full conversation. Ask follow-up questions, refer back to earlier answers, or drill deeper — it keeps the context.
Tips
Ask multi-part questions. Combine tools — "search for today's ETH price and calculate how much $500 is worth." Start a new thread anytime with New chat.
